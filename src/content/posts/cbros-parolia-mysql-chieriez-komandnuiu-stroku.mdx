Консоль ван лав. В ней можно делать всякое а можно просто запустить mc  и радоваться жизни ^_^ Но сегодня я поведаю вам, как сбросить пароль любого пользователя MySQL в этой самой консоли.

1. Останавливаем MySQL.
```bash
sudo service mysql stop
```

2. Прибиваем все запущенные `mysqld`.
```bash
sudo killall -9 mysqld
```

3. Запускаем `mysqld` в безопасном режиме.
```bash
sudo mysqld_safe --skip-grant-tables --skip-networking &
```

4. Подключаемся к MySQL.
```bash
mysql -u root
```

5. После успешного входа в систему выполните эту команду, чтобы изменить любой пароль.
```bash
FLUSH PRIVILEGES;
```

6. Для изменения пароля root пользователя выполняем следующее.
```bash
UPDATE mysql.user SET Password=PASSWORD('newpwd') WHERE User='root';
```
<InfoBlock title="Tip" variant="tip">
    Для MySQL > 5.7 используем эту команду.
    ```bash
    UPDATE mysql.user SET authentication_string=PASSWORD('newpwd') WHERE User='root';
    ```
</InfoBlock>

7. После выполняем следующую команду.
```bash
FLUSH PRIVILEGES;
```

8. Выходим из MySQL.
```bash
exit
```

9. Убиваем `mysqld_safe` и запускаем MySQL.
```bash
sudo killall mysqld_safe && sudo service mysql start
```
