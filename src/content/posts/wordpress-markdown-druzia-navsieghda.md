Небольшая инструкция о том как начать писать посты в [WordPress](https://ru.wordpress.org/) и оформлять их в разметке [Markdown](https://daringfireball.net/projects/markdown/syntax) или MD, как ее называют в сокращенном варианте. Из коробки stand-alone WordPress сайт не поддерживает этот тип разметки. Поэтому его надо установить.

## Как установить Markdown разметку в WordPress

Чтобы его установить есть два варианта.

Вариант первый. Установить плагин [JetPack](https://ru.wordpress.org/plugins/jetpack/), в котором есть большое количество настроек и возможностей. Одна из них как раз использование MD разметки.

Второй вариант. Чтобы не ставить огромный JetPack, установить отдельный плагин — который называется [JP Markdown](https://ru.wordpress.org/plugins/jetpack-markdown/). Активировать — и все готово. Теперь можно прямо в редакторе набирать текст оформляя его Markdown разметкой. В опубликованном варианте все будет обернуто нужными тегами.

Если вы используете WordPress с платформы [WordPress.com](https://ru.wordpress.com/) — то у вас уже установлен JetPack и уже есть эта возможность.
